# RoundLabs — Demo Flow (~75s)

Three environments. One shared engine. No fabricated data.

## 0–12s — Problem
Competitive players can see *what* happened, but most tools don't explain *which
decision* caused the mistake or what evidence was available at the moment.

## 12–24s — Value + recognition
Home screen: one match-moment input. Type a Valorant description → the recognition
banner shows **Detected: Valorant · deterministic · matched terms** (it never claims
AI when matching is deterministic). Architecture strip: 3 environments, 1 LabSpec,
1 renderer, 1 registry, game-specific manifests.

## 24–45s — Valorant verified lab (tactical-command)
Open the verified Valorant lab. Point out the **tactical command room** layout:
map-centred primary + "WHAT WE KNOW" intel rail, uppercase mono labels, timeline as
round progression. Facts are all `▲ Reported` (glyph+text, not colour-only); 6
unknowns visible; 3 alternatives with consequence/reasoning; selecting one changes
the reasoning; recommended decision; drill; measured processing time; Save Lesson.

## 45–60s — Cross-game proof (branch-analysis + race-strategy)
Back → open F1 → the same renderer produces a **race-strategy** workstation
(horizontal telemetry/strategy grammar, amber, "Awaiting verified race telemetry" —
honest empty state, no invented telemetry). Same for Pokémon's **branch-analysis**
board. One LabSpec, one renderer; the interface changes because the decision problem
changes — not just colours and labels.

## 60–75s — Close
Everything shown is real: verified provenance, 0 fabricated numeric facts, unknowns
marked unknown, honest empty states.
